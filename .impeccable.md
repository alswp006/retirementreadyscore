# RetirementReadyScore.impeccable.md

## Design Context

### Users
20-40대 직장인 및 자영업자 토스 사용자로, 은퇴 계획에 대한 불안감을 빠르게 해소하고 싶은 고객입니다. 현재 저축 현황과 지출 수준만으로 객관적인 은퇴 가능 시점을 알고 싶어 하며, 복잡한 금융 계산 없이 30초 내 결과를 받기를 원합니다.

### Brand Personality
**신뢰성 · 명확함 · 빠름**. 정확한 숫자 계산과 거짓 없는 결과 제시로 사용자의 금융 불안을 덜어주되, 장황하지 않고 직결된 답변을 제공합니다. 토스의 일관된 금융 정직성과 사용자 중심의 간결함을 유지합니다.

### Aesthetic Direction
TDS (Toss Design System) 준수. @toss/tds-mobile 컴포넌트 전용. 커스텀 미적 방향 없음.

### UX Principles
1. **즉각적 검증**: 입력 오류 시 명확한 필드별 도움말을 실시간 제시해 사용자가 유효한 값으로 빠르게 복구하도록 유도합니다.
2. **데이터 안전성**: 입력값 자동 저장으로 재진입 시 이전 데이터 복구 가능하며, 저장 실패 시에도 결과 계산과 네비게이션은 진행됩니다.
3. **광고 투명성**: 결과 공개를 위한 리워드 광고 요청 시 실패하면 즉시 대체 UI로 전환하고, 사용자가 재시도 또는 홈 복귀를 선택하게 합니다.
4. **정수 기반 계산**: 모든 금융 데이터는 정수로만 처리해 소수점 오류와 혼동을 제거합니다.
